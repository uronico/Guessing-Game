# Overview
Title:

Number Guessing Game

Description:

The Number Guessing Game is a simple, interactive console-based Java program designed for beginners learning programming fundamentals. The game challenges the player to guess a randomly generated number between 1 and 100 within a limited number of attempts. After each guess, the program provides helpful feedback, telling the player whether their guess is too high or too low, until they either guess correctly or run out of attempts.

Purpose:

I create this program to show what I learned about Java language. Such as using loops and conditionals, handling user input with scanner and working with the Random class.
